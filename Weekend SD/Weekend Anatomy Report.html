How to read this

Every number below is two-sided. For a mean of ~277 weekends, a t-statistic of magnitude |t| < 2 is statistically indistinguishable from zero — treat it as noise. |t| ≥ 2 clears the usual 5% bar for a single test; because this report runs ~28 tests, the honest threshold for "believe it" is nearer |t| ≥ 2.6 (see the honesty box). Green% = share of days that close above their open. "Close-to-close" return is yesterday-close → today-close, assigned to today's weekday.

4 · Does anything predict Monday? (c1–c4)

Definitions. Monday net move = base → Tuesday 00:00 Dubai close. "Continuation hit" = share of weekends where the target finished the same sign as the signal (a hit > 50% = momentum/continuation; < 50% = reversal). "dir-bet t" is the one-sample t of sign(signal)×target (the edge of actually trading the signal). Welch t compares the target mean on signal-up vs signal-down weekends.

Honesty box — sample size & multiple comparisons.
  • Sample: 277 weekends (~278 days per weekday). That is small: the standard error on a weekend mean is roughly the SD over √277 ≈ SD/16.6, so only sizeable effects can clear |t|≥2.
  • Tests run: ~28 (7 weekdays × 2 timezones for close-to-close means = 14, plus the two Sunday-window activity/direction tests, the four c1–c4 predictive tests, and the three c3 excursion tests). At a 5% level you should expect ~1–2 false positives with |t|≥2 by pure luck. Do not celebrate a lone |t|≈2.
  • Bonferroni-honest bar: for ~28 tests the family-wise 5% threshold is p ≈ 0.0018, i.e. |t| ≈ 3.1. Nothing here reaches that. The green/orange badges flag |t|≥2 and |t|≥2.6 only as signposts, not as proof.
  • What this report is: a map of where the weekend moves and which conditioning variables are worth carrying into a proper walk-forward test — not a validated edge. Treat the shortlist as hypotheses ranked by strength, all still to be earned out-of-sample.
