DT2 “Volatility-Surge Breakout” — Pine v6 (indicator + strategy)

The validated DT2 Volatility-Surge Breakout — DaviddTech’s library strategy C25-021-ATR-PERCENTILE-SURGE — rebuilt as a causal Pine v6 indicator plus a Strategy Tester twin with an identical signal core. BTC 4h ATR-percentile breakout: enter in the EMA20 trend direction when the ATR percentile surges up through a threshold, ride the expansion, exit when volatility normalises. Every decision, drawing and alert fires on confirmed, closed bars only — no repaint. This page is the copy of record and the paste source for both files.

Tested feed — chart this exact one: BINANCE:BTCUSDT perpetual, 4h. The edge was measured on Binance BTCUSDT-perp candles. The surge trigger is volatility-local: a different exchange’s ATR crosses the percentile threshold at slightly different bars and will not reproduce the record. Chart the same feed the backtest used. 2 · What it does & how it trades

  • Volatility percentile — atr = ta.atr(14); atrPct = ta.percentrank(atr, 50) = the percent of the previous 50 ATR values that are ≤ the current ATR (excludes the current bar), 0–100.
  • Surge trigger — atrPct crosses up through pctThr 80 on a closed 4h bar (atrPct > 80 and atrPct[1] ≤ 80).
  • Direction — close > EMA(20) → long, else short, evaluated at the signal bar. Longs and shorts both trade.
  • Fill — the open of the next 4h bar after the confirmed surge (causal). The stop geometry hangs off entryRef = the signal-bar close, not the fill price.
  • Moving hard stop — entryRef ∓ 2.5×ATR(14), re-armed every bar with the current ATR (entryRef fixed, ATR moves → the stop moves). Realized R is not pinned to −1: if ATR expands the stop is wider (loss > 1R); if it contracts the stop ratchets in (loss < 1R). No stop is active on the fill bar itself.
  • Vol-normalisation exit — atrPct < 50 on a closed bar → market close at the next open. This is the primary exit (the move matured) — it fires far more than the stop.
  • Time stop — bar_index − signalBar ≥ 42 bars → market close at the next open. The clock counts from the signal bar.
  • Exit priority — a pending market-close (vol-normalisation / time) executes at the bar open, chronologically first, and wins over the stop that same bar (the engine’s ambiguous-is-loss rule).
  • Sizing of record — 1%-of-equity risk against the initial stop distance (2.5×ATR at the signal bar); the strategy twin sizes the same way.

On-chart marks & the status panel

  • EMA(20) (orange) — the direction filter longs/shorts key off.
  • Entry triangle (green up / red down) prints at the signal bar — the actionable surge; the fill is the next 4h open. A small label at the fill bar quotes the entry (signal close) and initial stop.
  • Moving stop line (red step) runs for the life of each trade at the current entryRef ∓ 2.5×ATR — you can watch it ratchet as ATR changes.
  • Exit × — red at a moving-stop hit, blue at the atrPct<50 vol-normalisation close, orange at a time-stop close.
  • Status panel (top-right) — last-confirmed machine state: position, side, entry (ref), moving stop, ATR-percentile state (surge zone / cooling / normalised), bars in trade. A read-out, not a bar-anchored drawing.

The 1-bar Strategy-Tester lag caveat — read before judging the Tester’s P&L. The number of record is our Python backtest (+0.296R center), which fills at the open of the bar after the confirmed surge and detects the moving stop on 1m granularity. The Strategy-Tester twin submits its market order the moment that entry bar confirms, so TradingView fills it on the following open — about one 4h bar later than the engine — refreshes the moving stop each bar (strategy.exit) and closes the atrPct<50 / time exits at the next open, resolving the stop on 4h bars. 4 · Install on TradingView

  1. Open a BINANCE:BTCUSDT perpetual chart on the 4-hour timeframe — the exact feed and timeframe the edge was measured on.
  2. Load full history (scroll back to ~2019 or set a wide range) so the ATR percentile has room to seed — atrPct is undefined until roughly the first 14 + pctLen bars, which do not trade.
  3. Open Pine Editor, paste the indicator source (copy button below), Save, then Add to chart. It overlays the EMA, marks surges/exits and shows the status panel.
  4. For the Strategy Tester picture, add the strategy source as a second script — identical signals, plus the trade list and P&L (mind the 1-bar Tester-lag caveat above).
  5. To alert: right-click the chart → Add alert → Condition = this indicator → pick “LONG signal”, “SHORT signal”, “entry (either side)” or “exit” → Once per bar close. Entry alerts fire on the surge bar; enter at the next 4h open.
  6. Leave the signal knobs on their validated center defaults. They are exposed as inputs so you can inspect — not so you re-tune the edge.

//
// HOW FILLS WORK IN THE TESTER — the 1-bar Strategy-Tester lag caveat (read before
// judging its P&L): the record's fill is the OPEN of the bar AFTER the confirmed
// surge. This strategy submits its market order the moment that entry (fill) bar
// confirms, so the Tester fills on the FOLLOWING open — about one 4h bar later than
// the engine's fill. The MOVING stop is refreshed every bar (strategy.exit with the
// current entryRef -/+ slMult*ATR); the atrPct<50 and time-stop exits are
// strategy.close at the next open. Stop geometry hangs off entryRef = the SIGNAL-bar
// close (engine geometry); sizing is 1%-of-equity risk vs the INITIAL stop distance.
// TradingView also caps 4h history, so you see recent years, not the full 2019-2024
// record. The Tester is for SEEING the trades; the numbers of record are the backtest
// reports on the site (+0.296R center), NOT this chart's delayed-fill P&L.
//
// NO REPAINT (Igor's hard rules, audited): every decision, drawing, order and alert
// is gated by barstate.isconfirmed; chart timeframe ONLY (no request.security, varip,
// timenow, barstate.isrealtime); drawings APPEND-ONLY (label.new only; ZERO
// .set_*()/.delete()); nothing positioned at a past bar; the status panel is a
// last-confirmed read-out. Identical signal core to the indicator.
// =============================================================================

strategy("DT2 Vol-Surge — Strategy", overlay = true,
     initial_capital = 10000, currency = currency.USD,
     commission_type = strategy.commission.percent, commission_value = 0.05,
     slippage = 2, pyramiding = 0,
     max_lines_count = 500, max_labels_count = 500, max_boxes_count = 500)

// ----------------------------- Inputs ----------------------------------------
gS = "Signal (validated CENTER defaults — do not change to trade the edge)"
i_pctLen   = input.int(50,   "ATR-percentile lookback (pctLen)", minval = 2, group = gS)
i_pctThr   = input.float(80, "Surge threshold (pctThr)",         minval = 1, maxval = 100, step = 1, group = gS)
i_slMult   = input.float(2.5,"Stop ATR multiple (slMult)",       minval = 0.1, step = 0.1, group = gS)
i_maxBars  = input.int(42,   "Time stop (bars from signal, maxBars)", minval = 1, group = gS)
i_closeThr = input.float(50, "Vol-normalisation exit (atrPct < X)",   minval = 1, maxval = 100, step = 1, group = gS)
i_atrLen   = input.int(14,   "ATR length",                       minval = 1, group = gS)
i_emaLen   = input.int(20,   "EMA length (direction filter)",    minval = 2, group = gS)

gR = "Strategy sizing"
i_riskPct = input.float(1.0, "Risk per trade (% of equity)", minval = 0.01, maxval = 10.0, step = 0.1, group = gR)

gV = "Visuals"
i_showEma    = input.bool(true,  "EMA(20) direction line", group = gV)
i_showStop   = input.bool(true,  "Moving stop line while in a trade", group = gV)
i_showEntries= input.bool(true,  "Entry triangles + labels", group = gV)
i_showExits  = input.bool(true,  "Exit ×-marks", group = gV)
i_showState  = input.bool(true,  "Status panel (top-right)", group = gV)

// ----------------------------- Per-bar indicators ----------------------------
// Computed EVERY bar in global scope so ta.* history stays intact; only READ
// inside the confirmed block, so all DECISIONS use closed-bar values.
atrv    = ta.atr(i_atrLen)
ema20v  = ta.ema(close, i_emaLen)
atrPctv = ta.percentrank(atrv, i_pctLen)
surge   = not na(atrPctv) and not na(atrPctv[1]) and atrPctv > i_pctThr and atrPctv[1] <= i_pctThr

plot(i_showEma ? ema20v : na, "EMA(20)", color = color.new(color.orange, 0), linewidth = 1)

// ----------------------------- Persistent state ------------------------------
// Pure running state (no arrays, no look-back), exactly like the Python mirror
// pine_mirror_dt2.run_state_machine — forward only. IDENTICAL to the indicator.
var int   state      = 0
var float entryRef   = na
var float entryAtr   = na
var int   sigBar     = na
var int   fillBar    = na
var float activeStop = na
var int   pendClose  = 0

var int   pendDir    = 0
var float pendRef    = na
var float pendAtr    = na
var int   pendSig    = na

// per-bar event flags (reset each bar; drive marks + alerts + orders)
signalLong     = false
signalShort    = false
enterLongFill  = false
enterShortFill = false
exitStopEv     = false
exitNormEv     = false
exitTimeEv     = false
float exitPx   = na
float initStop = na

// process CONFIRMED bars only -> no repaint, deterministic
if barstate.isconfirmed
    // (1) manage an OPEN position on THIS bar ---------------------------------
    if state != 0
        if pendClose != 0
            exitPx := open
            if pendClose == 1
                exitNormEv := true
            else
                exitTimeEv := true
            state := 0
            activeStop := na
            pendClose := 0
        else if not na(activeStop) and ((state == 1 and low <= activeStop) or (state == -1 and high >= activeStop))
            exitStopEv := true
            exitPx := activeStop
            state := 0
            activeStop := na

    // (2) resolve a pending ENTRY armed at b-1 (only if flat now) --------------
    if pendDir != 0 and state == 0
        entryRef := pendRef
        entryAtr := pendAtr
        sigBar   := pendSig
        fillBar  := bar_index
        state    := pendDir
        activeStop := na
        pendClose  := 0
        initStop := pendDir == 1 ? entryRef - i_slMult * entryAtr : entryRef + i_slMult * entryAtr
        if pendDir == 1
            enterLongFill := true
        else
            enterShortFill := true
    pendDir := 0

    // (3) still open at the CLOSE of this bar -> arm stop + close-cond for b+1 -
    if state != 0
        activeStop := state == 1 ? entryRef - i_slMult * atrv : entryRef + i_slMult * atrv
        if (not na(atrPctv)) and atrPctv < i_closeThr
            pendClose := 1
        else if (bar_index - sigBar) >= i_maxBars
            pendClose := 2
    else
        // (4) flat -> detect a new surge at the CLOSE of this bar, arm for b+1 -
        if surge and not na(atrv) and i_slMult * atrv > 0
            pendDir := close > ema20v ? 1 : -1
            pendRef := close
            pendAtr := atrv
            pendSig := bar_index
            if pendDir == 1
                signalLong := true
            else
                signalShort := true

    // ---- causal, append-only drawings (nothing positioned at a past bar) ----
    if i_showEntries and enterLongFill
        label.new(bar_index, low, text = "LONG " + str.tostring(entryRef, format.mintick) + "  · stop " + str.tostring(initStop, format.mintick),
             style = label.style_label_up, color = color.new(color.green, 0), textcolor = color.white, size = size.tiny)
    if i_showEntries and enterShortFill
        label.new(bar_index, high, text = "SHORT " + str.tostring(entryRef, format.mintick) + "  · stop " + str.tostring(initStop, format.mintick),
             style = label.style_label_down, color = color.new(color.red, 0), textcolor = color.white, size = size.tiny)

    // ---- STRATEGY ORDERS (submitted at the fill bar -> Tester fills next open;
    //      the documented 1-bar Tester lag). Moving stop refreshed each bar; the
    //      vol-normalisation / time-stop exits flatten at the next open. -------
    if enterLongFill and not na(initStop)
        sd = math.abs(entryRef - initStop)
        q  = sd > 0 ? strategy.equity * i_riskPct / 100.0 / sd : 0.0
        if q > 0
            strategy.entry("DT2", strategy.long, qty = q, comment = "long surge")
    if enterShortFill and not na(initStop)
        sd = math.abs(entryRef - initStop)
        q  = sd > 0 ? strategy.equity * i_riskPct / 100.0 / sd : 0.0
        if q > 0
            strategy.entry("DT2", strategy.short, qty = q, comment = "short surge")
    // refresh the MOVING stop every bar the machine holds a position
    if state != 0 and not na(activeStop)
        strategy.exit("stop", "DT2", stop = activeStop, comment = "ATR moving stop")
    // market-close exits -> flatten at the next open
    if exitNormEv
        strategy.close("DT2", comment = "atrPct<50")
    if exitTimeEv
        strategy.close("DT2", comment = "time stop")

// ----------------------------- Marks (causal plots) --------------------------
plotshape(i_showEntries and signalLong,  title = "LONG signal (fill next open)",  style = shape.triangleup,   location = location.belowbar, color = color.new(color.green, 0), size = size.tiny)
plotshape(i_showEntries and signalShort, title = "SHORT signal (fill next open)", style = shape.triangledown, location = location.abovebar, color = color.new(color.red, 0),   size = size.tiny)
plotshape(i_showExits and exitStopEv ? exitPx : na, title = "Stop exit",              style = shape.xcross, location = location.absolute, color = color.new(color.red, 0),    size = size.tiny)
plotshape(i_showExits and exitNormEv ? exitPx : na, title = "Vol-normalisation exit", style = shape.xcross, location = location.absolute, color = color.new(color.blue, 0),   size = size.tiny)
plotshape(i_showExits and exitTimeEv ? exitPx : na, title = "Time-stop exit",          style = shape.xcross, location = location.absolute, color = color.new(color.orange, 0), size = size.tiny)

plot(i_showStop and state != 0 ? activeStop : na, "Moving stop", color = color.new(color.red, 0), style = plot.style_linebr, linewidth = 1)

// ----------------------------- Alerts ----------------------------------------
eitherEntry = signalLong or signalShort
anyExit     = exitStopEv or exitNormEv or exitTimeEv
alertcondition(signalLong,  title = "DT2 Vol-Surge — LONG signal",
     message = "DT2 Vol-Surge: LONG volatility surge confirmed on {{ticker}} 4h at {{time}} — enter at the NEXT 4h open. Moving stop = entryRef - 2.5xATR(14). Chart BINANCE:BTCUSDT perp.")
alertcondition(signalShort, title = "DT2 Vol-Surge — SHORT signal",
     message = "DT2 Vol-Surge: SHORT volatility surge confirmed on {{ticker}} 4h at {{time}} — enter at the NEXT 4h open. Moving stop = entryRef + 2.5xATR(14). Chart BINANCE:BTCUSDT perp.")
alertcondition(eitherEntry, title = "DT2 Vol-Surge — entry (either side)",
     message = "DT2 Vol-Surge entry signal confirmed on {{ticker}} 4h at {{time}} — fill at the next 4h open.")
alertcondition(anyExit, title = "DT2 Vol-Surge — exit",
     message = "DT2 Vol-Surge: position exit on {{ticker}} 4h at {{time}} (moving ATR stop, atrPct<50 vol-normalisation, or time stop).")

// ----------------------------- Status panel ----------------------------------
var table st = table.new(position.top_right, 2, 6,
     bgcolor = color.new(color.black, 15), border_width = 1,
     border_color = color.new(color.gray, 60), frame_width = 1,
     frame_color = color.new(color.gray, 40))

if i_showState and barstate.islast
    posTxt  = state == 1 ? "LONG open" : state == -1 ? "SHORT open" : "flat"
    sideTxt = state == 1 ? "long" : state == -1 ? "short" : "—"
    entTxt  = state != 0 and not na(entryRef) ? str.tostring(entryRef, format.mintick) : "—"
    stopTxt = state != 0 and not na(activeStop) ? str.tostring(activeStop, format.mintick) : "—"
    pctTxt  = na(atrPctv) ? "—" : str.tostring(atrPctv, "#.0") +
         (atrPctv >= i_pctThr ? " · surge zone" : atrPctv < i_closeThr ? " · normalised (exit)" : " · cooling")
    barsTxt = state != 0 and not na(sigBar) ? str.tostring(bar_index - sigBar) : "—"
    _hd = color.new(color.silver, 20)
    _tx = color.white
    table.cell(st, 0, 0, "DT2 position", text_color = _hd, text_size = size.small, text_halign = text.align_left)
    table.cell(st, 1, 0, posTxt,         text_color = _tx, text_size = size.small, text_halign = text.align_left)
    table.cell(st, 0, 1, "side",         text_color = _hd, text_size = size.small, text_halign = text.align_left)
    table.cell(st, 1, 1, sideTxt,        text_color = _tx, text_size = size.small, text_halign = text.align_left)
    table.cell(st, 0, 2, "entry (ref)",  text_color = _hd, text_size = size.small, text_halign = text.align_left)
    table.cell(st, 1, 2, entTxt,         text_color = _tx, text_size = size.small, text_halign = text.align_left)
    table.cell(st, 0, 3, "moving stop",  text_color = _hd, text_size = size.small, text_halign = text.align_left)
    table.cell(st, 1, 3, stopTxt,        text_color = _tx, text_size = size.small, text_halign = text.align_left)
    table.cell(st, 0, 4, "ATR pctile",   text_color = _hd, text_size = size.small, text_halign = text.align_left)
    table.cell(st, 1, 4, pctTxt,         text_color = _tx, text_size = size.small, text_halign = text.align_left)
    table.cell(st, 0, 5, "bars in trade",text_color = _hd, text_size = size.small, text_halign = text.align_left)
    table.cell(st, 1, 5, barsTxt,        text_color = _tx, text_size = size.small, text_halign = text.align_left)
