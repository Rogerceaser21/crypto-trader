Annie Monday Fade — Strategy V1

The strategy() version of the validated Monday fade — add it to a BTCUSDT perp 15m chart and TradingView's Strategy Tester shows every trade: entry/exit arrows on the candles, the trade list, win/loss and P&L. Signal logic is line-identical to the Monday Fade indicator (87/91 validated entries exact to the bar). Validated stats: IS +0.382R over 91 trades, locked OOS +0.54R over 28.

1 · What the Tester shows — and what it doesn't

Read this before judging the P&L. The winning Monday spike is only knowable once the early week closes (~Tue 17:00 NY). We measured the alternative on our 5-year data: a strategy that commits in real time to the running spike reproduces the validated trades plus 147 phantom trades the validated system never took — rejected. So this strategy locks at the early-window close:

  • Setups completing after the lock fill on the true next-bar open (20 of the 91 historical entries).
  • Setups completing before the lock fill at the first bar after the lock — historically ~22h later than the engine's fill (71 of 91). Their stop/target LEVELS still use the original engine geometry, and the “engine bar” label marks the true entry candle.

Consequences: the Tester's trade LIST and levels mirror the validated system; the P&L of the delayed fills does not. The authoritative economics are the site's backtest reports. TradingView also caps 15m history (~20k bars), so you'll see the recent months, not all of 2019–2024.

3 · Install

  1. Open a BTCUSDT perp chart on 15m.
  2. Pine Editor → Open → New blank strategy (or blank indicator — the script declares itself).
  3. Select all (Cmd/Ctrl + A), delete, paste the source below (button provided).
  4. Save as Annie Monday Fade Strategy V1 → Add to chart.
  5. Open the Strategy Tester panel (bottom) → “List of trades” to walk every trade; click a row to jump the chart there.
  6. Optional: keep the indicator version on the same chart — its arrows mark the engine's true entry bars.

4 · Full source — copy & paste

Select all inside the box, or use the button, then paste into a blank Pine script. 
//@version=6
// =============================================================================
// Annie Monday Fade — Strategy V1
// -----------------------------------------------------------------------------
// strategy() version of the validated Annie Monday fade so TradingView's
// Strategy Tester shows every trade: entry/exit arrows, trade list, P&L.
// Signal logic = IDENTICAL to "Annie Monday Fade — LIVE" (the indicator),
// which matches the validated engine 87/91 entries exact-to-the-bar (95.6%).
// Validated stats (our engine, 5y in-sample): 91 Monday trades, 54.9% WR,
// +0.382R, PF 2.05 (t=2.98); locked OOS 2025-26: +0.54R over 28 trades.
//
// HOW FILLS WORK IN THIS STRATEGY (read before judging Tester P&L):
//   The winning Monday spike is only KNOWABLE once the early window closes
//   (~Tue 17:00 NY). Design measured on 2026-07-13 (f10_streaming_mirror.py):
//   committing to the running extreme in real time reproduces the validated
//   trades BUT also takes 147 phantom trades the validated system never took
//   -> rejected. This strategy therefore LOCKS at the early-window close:
//     • setups completing AFTER the lock fill on the true next-bar open
//       (20 of the 91 historical entries);
//     • setups completing BEFORE the lock fill at the first bar after the
//       lock — historically ~22h later than the engine's fill (71 of 91).
//   Stop and target LEVELS always use the ORIGINAL engine geometry (computed
//   from the true entry bar), so what you see on the chart is the validated
//   trade plan; only the delayed fills' P&L differs from the reports.
//   The authoritative economics live in the backtest reports on the site —
//   the Tester here is for SEEING the trades, not re-deriving the stats.
//   Also: TradingView caps 15m history (~20k bars), so you will see the
//   recent months of trades, not the whole 2019-2024 record.
//
// TP is an INPUT (default 1.9R = the validated baseline). Flip it to 2.5 to
// see the Exit-RR-Sweep finding (+0.506R vs +0.382R on the frozen 91).
//
// PVSRA 200%-class vector: volume >= 2.0x the average of the PRIOR 10 candle
// volumes (ta.sma(volume[1],10)); green if close>=open, red if close<open.
// Chart: BTCUSDT perp, 15m.
// =============================================================================

strategy("Annie Monday Fade — Strategy V1", overlay = true,
     initial_capital = 100000, currency = currency.USD,
     commission_type = strategy.commission.percent, commission_value = 0.05,
     slippage = 2, max_lines_count = 500, max_labels_count = 500, max_boxes_count = 500)

// ----------------------------- Inputs ----------------------------------------
gA = "Anchors (Igor's weekly model)"
i_tz       = input.string("America/New_York", "Anchor timezone", group = gA)
i_openHour = input.int(17, "Week open hour (Sunday, NY)", minval = 0, maxval = 23, group = gA)
i_friHour  = input.int(8,  "Friday hard-cutoff hour (NY)", minval = 0, maxval = 23, group = gA)
i_earlyFrac= input.float(0.286, "Early-window fraction of week (Sun/Mon/Tue)", minval = 0.1, maxval = 0.6, step = 0.001, group = gA)

gD = "Detector knobs (validated defaults — do not change to trade the edge)"
i_emaLen   = input.int(50, "50-EMA length", minval = 2, group = gD)
i_volLb    = input.int(10, "PVSRA volume lookback (prior N)", minval = 2, group = gD)
i_vecMult  = input.float(2.0, "200%-class vector threshold (x vol avg)", minval = 1.0, step = 0.1, group = gD)
i_minSpike = input.float(0.10, "Min spike beyond box (%)", minval = 0.0, step = 0.01, group = gD)
i_touchTol = input.float(0.15, "50-EMA wick-touch tolerance (%)", minval = 0.0, step = 0.01, group = gD)
i_crossTol = input.float(0.10, "50-EMA body-cross tolerance (%)", minval = 0.0, step = 0.01, group = gD)
i_span     = input.int(320, "Max bars L1 -> retest", minval = 1, group = gD)
i_stopBuf  = input.float(0.05, "Stop buffer beyond extreme (%)", minval = 0.0, step = 0.01, group = gD)
i_rr       = input.float(1.9, "TP (R multiple) — try 2.5 (Exit RR Sweep winner)", minval = 0.1, step = 0.1, group = gD)
i_mondayOnly = input.bool(true, "Monday-spike only (the validated edge)", group = gD)

gR = "Strategy sizing"
i_riskPct  = input.float(1.0, "Risk per trade (% of equity)", minval = 0.01, maxval = 10.0, step = 0.1, group = gR)

gV = "Visuals"
i_showBox    = input.bool(true,  "Weekend box", group = gV)
i_showPeak   = input.bool(true,  "Spike-peak marker", group = gV)
i_showL1     = input.bool(true,  "L1 vector highlight", group = gV)
i_showRetest = input.bool(true,  "Retest marker", group = gV)
i_showEntry  = input.bool(true,  "True-entry-bar arrow (engine timing)", group = gV)
i_showStopTP = input.bool(true,  "Stop / TP lines", group = gV)
i_showEma    = input.bool(true,  "50-EMA", group = gV)

// derived tolerances (fractions)
touchTol = i_touchTol / 100.0
crossTol = i_crossTol / 100.0
minSpike = i_minSpike / 100.0
stopBuf  = i_stopBuf  / 100.0

DAY_MS = 86400000

// ----------------------------- Per-bar indicators ----------------------------
ema50 = ta.ema(close, i_emaLen)
avgVol   = ta.sma(volume[1], i_volLb)
volRatio = na(avgVol) or avgVol == 0 ? 0.0 : volume / avgVol
is200    = volRatio >= i_vecMult
isUp     = close >= open

plot(i_showEma ? ema50 : na, "50 EMA", color = color.new(color.orange, 0), linewidth = 1)

// ----------------------------- Week time anchors -----------------------------
_dow  = dayofweek(time, i_tz)          // 1=Sunday .. 7=Saturday
_yr   = year(time, i_tz)
_mo   = month(time, i_tz)
_dy   = dayofmonth(time, i_tz)
_today0     = timestamp(i_tz, _yr, _mo, _dy, i_openHour, 0)
_daysBack   = _dow - 1                  // Sun=0 .. Sat=6
_approxSun  = _today0 - _daysBack * DAY_MS
weekOpenTs  = timestamp(i_tz, year(_approxSun, i_tz), month(_approxSun, i_tz), dayofmonth(_approxSun, i_tz), i_openHour, 0)
if time < weekOpenTs
    _approxSun2 = _today0 - (_daysBack + 7) * DAY_MS
    weekOpenTs := timestamp(i_tz, year(_approxSun2, i_tz), month(_approxSun2, i_tz), dayofmonth(_approxSun2, i_tz), i_openHour, 0)

weekEndTs  = weekOpenTs + 7 * DAY_MS
earlyEndTs = weekOpenTs + int(i_earlyFrac * 7.0 * DAY_MS)
_friApprox = weekOpenTs + 5 * DAY_MS
friEndTs   = timestamp(i_tz, year(_friApprox, i_tz), month(_friApprox, i_tz), dayofmonth(_friApprox, i_tz), i_friHour, 0)

newWeek = weekOpenTs != weekOpenTs[1]

// ----------------------------- Persistent week state -------------------------
var array<float> wkO = array.new_float()
var array<float> wkH = array.new_float()
var array<float> wkL = array.new_float()
var array<float> wkC = array.new_float()
var array<float> wkE = array.new_float()
var array<int>   wkT = array.new_int()
var array<bool>  wk2 = array.new_bool()
var array<bool>  wkU = array.new_bool()

var float boxTop = na
var float boxBot = na
var float pendTop = na
var float pendBot = na

var float runHigh = na
var float runLow  = na
var int   extHiWi = 0
var int   extLoWi = 0

var bool  locked   = false
var bool  resolved = false
var int   fadeDir  = 0          // 1 long, -1 short, 0 none
var int   extWi    = 0
var bool  l1Found  = false
var int   l1Wi     = 0
var float l1Low    = na
var float l1High   = na
var bool  c2latch  = false
var int   scanWi   = 0

entryFired = false

// process only CONFIRMED bars -> no repaint, deterministic
if barstate.isconfirmed
    // --- New week bootstrap ---
    if newWeek
        boxTop := pendTop
        boxBot := pendBot
        pendTop := na
        pendBot := na
        array.clear(wkO)
        array.clear(wkH)
        array.clear(wkL)
        array.clear(wkC)
        array.clear(wkE)
        array.clear(wkT)
        array.clear(wk2)
        array.clear(wkU)
        runHigh := na
        runLow  := na
        extHiWi := 0
        extLoWi := 0
        locked   := false
        resolved := false
        fadeDir  := 0
        l1Found  := false
        c2latch  := false
        scanWi   := 0
        if i_showBox and not na(boxTop) and not na(boxBot)
            box.new(weekOpenTs - 2 * DAY_MS, boxTop, weekOpenTs, boxBot, xloc = xloc.bar_time,
                 border_color = color.new(color.gray, 40), bgcolor = color.new(color.gray, 90))

    array.push(wkO, open)
    array.push(wkH, high)
    array.push(wkL, low)
    array.push(wkC, close)
    array.push(wkE, ema50)
    array.push(wkT, time)
    array.push(wk2, is200)
    array.push(wkU, isUp)
    curWi = array.size(wkH) - 1

    // --- weekend box accumulation (this week's following weekend) ---
    if time >= weekOpenTs + 5 * DAY_MS and time < weekEndTs
        pendTop := na(pendTop) ? close : math.max(pendTop, close)
        pendBot := na(pendBot) ? close : math.min(pendBot, close)

    // --- early-window running extreme ---
    if time <= earlyEndTs
        if na(runHigh) or high > runHigh
            runHigh := high
            extHiWi := curWi
        if na(runLow) or low < runLow
            runLow := low
            extLoWi := curWi

    // --- LOCK at the early-window close ---
    if not locked and not resolved and time > earlyEndTs
        locked := true
        if na(boxTop) or na(boxBot) or boxTop <= boxBot or na(runHigh) or na(runLow)
            resolved := true
        else
            upExc   = (runHigh - boxTop) / boxTop
            downExc = (boxBot - runLow) / boxBot
            if math.max(upExc, downExc) < minSpike
                resolved := true
            else
                if upExc >= downExc
                    fadeDir := -1
                    extWi   := extHiWi
                else
                    fadeDir := 1
                    extWi   := extLoWi
                extT  = array.get(wkT, extWi)
                extHr = hour(extT, i_tz)
                extDw = dayofweek(extT, i_tz)
                isMon = (extHr < i_openHour and extDw == 2) or (extHr >= i_openHour and extDw == 1)
                if i_mondayOnly and not isMon
                    resolved := true
                else
                    scanWi := extWi
                    if i_showPeak
                        pkPrice = fadeDir == -1 ? runHigh : runLow
                        label.new(array.get(wkT, extWi), pkPrice,
                             text = fadeDir == -1 ? "▲ peak" : "▼ peak",
                             xloc = xloc.bar_time, style = fadeDir == -1 ? label.style_label_down : label.style_label_up,
                             color = color.new(color.gray, 20), textcolor = color.white, size = size.tiny)

    // --- RESOLVE scan: L1 -> retest -> entry (causal, forward, c1&c2) ---
    if locked and not resolved and scanWi <= curWi
        for wi = scanWi to curWi
            if resolved
                break
            H = array.get(wkH, wi)
            L = array.get(wkL, wi)
            O = array.get(wkO, wi)
            C = array.get(wkC, wi)
            E = array.get(wkE, wi)
            V2 = array.get(wk2, wi)
            UP = array.get(wkU, wi)
            emaOk = not na(E) and E != 0
            if not l1Found
                brk = false
                if emaOk and V2
                    if fadeDir == 1 and UP and C >= E and O <= E * (1 + crossTol)
                        brk := true
                    if fadeDir == -1 and not UP and C <= E and O >= E * (1 - crossTol)
                        brk := true
                if brk
                    l1Found := true
                    l1Wi    := wi
                    l1Low   := L
                    l1High  := H
                    c2latch := false
                    if i_showL1
                        box.new(array.get(wkT, wi), H, array.get(wkT, wi) + 15 * 60000, L,
                             xloc = xloc.bar_time, border_color = color.new(color.yellow, 0),
                             bgcolor = color.new(color.yellow, 80))
            else if wi > l1Wi
                if wi >= l1Wi + i_span
                    resolved := true
                else if emaOk
                    if fadeDir == 1 and C < E * (1 - crossTol)
                        resolved := true
                    else if fadeDir == -1 and C > E * (1 + crossTol)
                        resolved := true
                    else
                        if H >= l1Low and L <= l1High
                            c2latch := true
                        touched = fadeDir == 1 ? L <= E * (1 + touchTol) : H >= E * (1 - touchTol)
                        bodyOk  = fadeDir == 1 ? math.min(O, C) >= E * (1 - crossTol) : math.max(O, C) <= E * (1 + crossTol)
                        if touched and bodyOk and c2latch
                            resolved := true
                            entryWi   = wi + 1
                            hasNext   = entryWi < array.size(wkO)
                            entT      = hasNext ? array.get(wkT, entryWi) : na
                            entryRef  = hasNext ? array.get(wkO, entryWi) : na
                            wick = fadeDir == -1 ? array.get(wkH, 0) : array.get(wkL, 0)
                            for k = 0 to wi
                                wick := fadeDir == -1 ? math.max(wick, array.get(wkH, k)) : math.min(wick, array.get(wkL, k))
                            stopOk = hasNext and (fadeDir == -1 ? wick > entryRef : wick < entryRef)
                            valid  = hasNext and entT < friEndTs and stopOk
                            if valid
                                // ORIGINAL engine geometry (from the true entry bar)
                                stopP  = fadeDir == -1 ? wick * (1 + stopBuf) : wick * (1 - stopBuf)
                                sd     = math.abs(entryRef - stopP)
                                tpP    = fadeDir == -1 ? entryRef - i_rr * sd : entryRef + i_rr * sd
                                dirCol = fadeDir == -1 ? color.red : color.green
                                if i_showRetest
                                    label.new(array.get(wkT, wi), array.get(wkE, wi), text = "retest",
                                         xloc = xloc.bar_time, style = label.style_label_center,
                                         color = color.new(color.blue, 30), textcolor = color.white, size = size.tiny)
                                if i_showEntry
                                    label.new(entT, entryRef, text = fadeDir == -1 ? "SHORT (engine bar)" : "LONG (engine bar)",
                                         xloc = xloc.bar_time, style = fadeDir == -1 ? label.style_label_down : label.style_label_up,
                                         color = color.new(dirCol, 0), textcolor = color.white, size = size.small)
                                if i_showStopTP
                                    line.new(entT, stopP, friEndTs, stopP, xloc = xloc.bar_time,
                                         color = color.new(color.red, 0), width = 1, style = line.style_dashed)
                                    line.new(entT, tpP, friEndTs, tpP, xloc = xloc.bar_time,
                                         color = color.new(color.green, 0), width = 1, style = line.style_dashed)
                                    label.new(friEndTs, stopP, "stop", xloc = xloc.bar_time, style = label.style_label_left,
                                         color = color.new(color.red, 30), textcolor = color.white, size = size.tiny)
                                    label.new(friEndTs, tpP, str.tostring(i_rr, "#.#") + "R TP", xloc = xloc.bar_time, style = label.style_label_left,
                                         color = color.new(color.green, 30), textcolor = color.white, size = size.tiny)
                                entryFired := true
                                // ---- STRATEGY ORDERS (fill = next bar open) --
                                // risk-normalized size: i_riskPct% of equity per 1R
                                qty = strategy.equity * i_riskPct / 100.0 / sd
                                if qty > 0
                                    strategy.entry("fade", fadeDir == 1 ? strategy.long : strategy.short, qty = qty,
                                         comment = fadeDir == 1 ? "fade LONG" : "fade SHORT")
                                    strategy.exit("exit", "fade", stop = stopP, limit = tpP)
        scanWi := curWi + 1

    // --- Friday hard cap: flat before the weekend, always ---
    if strategy.position_size != 0 and time >= friEndTs
        strategy.close("fade", comment = "friday cap")

// ----------------------------- Alert -----------------------------------------
alertcondition(entryFired, title = "Annie Monday Fade — entry",
     message = "Annie Monday Fade Strategy V1 entry confirmed on BTCUSDT 15m ({{ticker}} @ {{time}})")

plotshape(entryFired, title = "entry (signal bar)", style = shape.diamond,
     location = location.bottom, color = color.new(color.aqua, 0), size = size.tiny)
